# RestaurantSeeker

Android sample App using Kotlin and Jetpack components to sort through a list of restaurants from a local json file.

Main libraries used:
- Live Data
- ViewModel
- Data Binding 
- Room
- Kotlin Coroutines

### Testing
The Unit tests and UI tests are in the respective folders ('test' and 'androidTest').
